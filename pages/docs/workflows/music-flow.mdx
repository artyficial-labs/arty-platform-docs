## Music Workflow

To create a music generation workflow you will need 2 nodes

- Music Generation Node 
- Music Preview Node 

![Image](https://res.cloudinary.com/dvu6ueq2b/image/upload/v1755727603/Docs%20images/music-workflow_kbeive.png)

### Music Generation Node 

Generates music from text descriptions. Features extensive customization with model selection, duration, output format and advanced parameters like temperature, classifier-free guidance, and normalization strategies.

### Music Preview Node 

Displays and plays audio files from the Music Node or other sources. Provides audio playback controls and download functionality.
