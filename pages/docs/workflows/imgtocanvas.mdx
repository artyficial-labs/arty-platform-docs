## Image-to-canvas Workflow

To create an image-to-canvas generation workflow you will need 3 nodes

- Image Generation Node 
- Video Preview Node
- Design Whiteboard Node

![Image](/imgtocanvas.png)

### Image Generation 

Refer to the Image Generation Node for more details on how to configure your image generation for the best quality

### Image Preview Node

Set your your Image Preview Node to "General Image" use to import the image to your canvas

### Design Whiteboard Node

Your image should be visible on your whiteboard once you have generated your image. You will also notice that a left sidebar with the tools needed for your canvas node shows up on your left.
