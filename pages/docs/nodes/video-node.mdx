## Video Generation Node

Creates videos from iamges or text prompts. In order to create a flow with this node you will need to either connect an image preveiew node to the video node and a prompt or just prompt away without an image. The generated videos display within the node for preview.

![Image](/video-node.png)


## Input prompts

- First Frame 
- Subject Reference

## Connectors

- Image Preview Node
- Video Preview Node(Output port)
